# codex-demo

A tiny beginner-friendly web sample that demonstrates HTML, CSS, and JavaScript.

## Getting started

1. Open `index.html` in your browser.
2. Click the **Say hello** button to see JavaScript in action.
3. Use the counter buttons to increase or reset the number.
4. Edit the text or styles and refresh to see your changes.

## Files

- `index.html` - The page structure and content.
- `styles.css` - Visual styling for the sample.
- `script.js` - A small interactive greeting.
