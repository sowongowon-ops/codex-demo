# Tetris Arcade

This project is a simple browser-based Tetris game built with HTML, CSS, and JavaScript.

## Features

- Playable Tetris board with rotating pieces
- Score, lines, and level tracking
- Next-piece preview
- Pause and restart controls

## Controls

- **Left/Right Arrow**: Move piece
- **Up Arrow**: Rotate piece
- **Down Arrow**: Soft drop
- **Space**: Hard drop
- **P**: Pause/Resume

## Run locally

```bash
python -m http.server 8000
```

Then open <http://localhost:8000> in your browser.
